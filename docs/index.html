Kiriは、複数エンジンの評価値乖離を核心に置く将棋解析プロジェクトです。水匠5とtanukiが同じ局面を異なる値で評価するとき、その差分の中に「一方のエンジンが見えていない何か」が潜んでいます。

棋譜をKIFとして入力すると、Mac mini上のやねうら王が全手をdepth 12で解析。Chart.jsによるインタラクティブなグラフと、Claudeによる局面解説を自動生成します。
